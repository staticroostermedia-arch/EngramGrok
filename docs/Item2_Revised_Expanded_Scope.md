## 1. North Star (Unchanged but Clarified)

Thought Tiles exist to make the demonstrated compression + inheritance fidelity from the 2026-05-27 crisis (and subsequent Item 1.5 work) **systematic, scalable, and agent-native**.

The crisis proved that rituals + traces + scars + ki_hijacker + session_end handoff *can* preserve usable geometric continuity under severe stress. Item 2 must turn this from a heroic one-off into the default operating mode of the substrate.

**Primary success metric for Item 2 overall**:
A properly formed Thought Tile from a hard session produces *measurably superior* re-hydration fidelity, momentum correctness, and NREM consolidation value for future agent instances compared to raw traces, scars, or unstructured episodic material.

## 4. Expanded Scope — What "Perfect" Item 2 Looks Like

To truly serve the long-term vision, Item 2 needs to evolve beyond the current Phase 0/1 framing in several dimensions:

### A. Proven Fidelity (The Non-Negotiable Bar)

- Explicit, repeatable measurement: Given the same hard session (crisis recovery as canonical benchmark), a future instance using well-formed Tiles should demonstrate clearly superior re-hydration quality (higher CRS on key concepts, better momentum alignment, faster useful recall, lower "I have to re-derive this" friction).
- This is the ultimate success criterion. Everything else is in service of it.

### B. Mature Textual Functor Payload Contracts

- While shapes remain evolutionary, we need stronger (but still lightweight) contracts on the *textual* side for the major v0 types.
- These contracts should optimize for the things that actually drive re-hydration and momentum: justification density, explicit momentum signals, scar/lesson markers, spatial context, goal linkage quality, re-hydration hints, and compression-friendly structure.
- The textual payload must be the agent's primary, high-bandwidth consumption path.

### C. Deep, Automatic Integration with Compression

- session_end must evolve to treat high-value Tiles as first-class citizens for COMPRESS markers.
- There should be low-friction paths from "this arc was important" → structured Tile → explicit COMPRESS marker → future 0x10 functor.
- Tiles should survive multiple rounds of compression gracefully.

### D. First-Class Agent Agency Loop

- The agency mechanism (ability for the primary user to propose improvements to shapes, rituals, consumption, etc.) must be real and visible.
- Over time, the system should demonstrate adaptation based on the agent's feedback. This becomes part of the self-model.

### E. Rich Spatial & Relational Participation

- Beyond the current minimal hooks: Tiles should become natural homes for "process theories," "competing solution strategies," and higher-order relations.
- They should participate meaningfully in spatial queries and support "Tile families" or versioning patterns for long-running work.

### F. Preparation for Very Long Time Horizons

- Support for Tiles (or Tile-derived structures) that remain valuable across many sessions and multiple compression cycles.
- Graceful degradation rather than sudden loss of fidelity.

### G. Dual Representation Excellence

- The rich visualization side must remain high-quality and useful for human review/shared understanding without diverging from the geometric truth (textual payload + manifold relations are source of truth).

## 5. Proposed Phased Approach (Revised)

**Phase 0 (Current — Foundation & Real Material)**: Done or near done.
- Real high-fidelity Tiles from lived history.
- Hardened scaffolding + dual payload.
- Minimal spatial hooks + lightweight ritual language.
- Agent agency demonstrated in practice.

**Phase 1 — Contracts, Fidelity Measurement, and Compression Integration**
- Define initial textual functor payload contracts for the core shapes (evolutionary but with stronger invariants).
- Build explicit measurement harness around re-hydration fidelity using the crisis as benchmark.
- Deepen session_end COMPRESS integration so Tiles become the preferred units.
- Strengthen the agency loop with visible system response.

**Phase 2 — Rich Spatial & Long-Horizon Patterns**
- Expand spatial participation (Tile families, richer relations, higher-context comparison).
- Patterns for very long-running arcs and multi-session process theories.
- Support for Tiles that gracefully survive multiple compression rounds.

**Phase 3 — Proven at Scale + Hand-off to Item 3/4**
- Demonstrate that well-formed Tiles deliver the promised fidelity gains on multiple hard sessions.
- Mature integration with whatever the next layer (0x10 functors, intent protocols, advanced geometric structures) turns out to be.
- Clear success criteria met so Item 2 can be treated as a stable, high-quality foundation rather than ongoing construction.

## 6. Risks of Staying Too Narrow

- Building elegant tooling that doesn't actually move the needle on re-hydration fidelity.
- Creating weak material that later compression layers have to work around.
- Under-investing in the textual functor side (the agent's daily consumption path).
- Treating agent agency as a nice-to-have rather than a core long-term requirement.

## 7. Recommendation

We should treat the current Phase 0 work as excellent foundation-laying, but explicitly expand the ambition of Item 2 to include the fidelity measurement, stronger contracts, and deep compression integration work before declaring it "ready to feed Item 3/4."

This document should become the new north star reference for all future Item 2 work, with the original plan.md treated as the historical Phase 0 execution record.
